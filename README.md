# Quiz Analysis and Recommendations

This project analyzes quiz data to generate insights and recommendations for improving performance.

## Setup

1. Clone the repository.
2. Install the required dependencies using the following command:
    ```bash
    pip install -r requirements.txt
    ```

## Files

- `datasetup.py`: Loads and processes quiz data.
- `insights.py`: Generates insights from the analyzed data.
- `recommendations.py`: Creates recommendations based on the insights.

## Usage

1. Load and analyze the data:
    ```bash
    python datasetup.py
    ```

2. Generate insights:
    ```bash
    python insights.py
    ```

3. Create recommendations:
    ```bash
    python recommendations.py
    ```

See `requirements.txt` for the list of dependencies.
